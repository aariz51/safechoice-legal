# SafeChoice Legal Pages

Public legal pages for the SafeChoice app.

This repository hosts simple static HTML pages for app support and compliance needs.

## Pages

- `index.html`
- `privacy.html`
- `terms.html`

## Tech stack

- HTML
- GitHub Pages

## Purpose

Mobile and web apps often need public privacy and terms URLs for app review, onboarding, and account-support flows. This repo keeps those pages versioned and easy to host.

## Deployment

GitHub Pages is enabled for this repository.
